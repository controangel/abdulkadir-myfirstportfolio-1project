# 🌐 AWS Static Website Portfolio

This project is a personal portfolio website hosted using Amazon S3 under the AWS Free Tier. It demonstrates how to deploy a secure, scalable, and cost-free static website using cloud services.

---

## 🚀 Live Demo

🔗 http://your-bucket-name.s3-website-region.amazonaws.com

---

## 📌 Project Overview

This project showcases a simple CV/portfolio website built with HTML and CSS and deployed on AWS.

It highlights:

* Cloud fundamentals using AWS
* Static website hosting without servers
* Secure public access configuration

---

## 🧰 Technologies Used

* Amazon S3 (Static Website Hosting)
* AWS IAM (Access Control)
* HTML5
* CSS3

---

## 🏗️ Architecture

User → Browser → Amazon S3 (Static Website Hosting)

---

## ⚙️ Deployment Steps

1. Created an S3 bucket
2. Uploaded website files (index.html, style.css)
3. Enabled static website hosting
4. Configured bucket policy for public access
5. Accessed website via S3 endpoint URL

---

## 💰 Cost Optimization

This project is designed to run within AWS Free Tier limits:

* No use of EC2 or paid services
* Minimal storage and requests
* No data transfer-heavy operations
